## Overview

AI Student Progress Management System is a comprehensive academic application designed for Admin, Teacher, and Student workflows. It streamlines school operations by tracking grades, managing assignments, calculating attendance, and rendering visual reports.

## Key Achievements & Implementation

- **Role-Based Access Control**: Built full-stack authentication workflows supporting three separate access roles (Admin, Teacher, Student) using JWT and database roles.
- **Data Analytics Dashboard**: Developed interactive charts and metrics dashboards for student grading distribution, average marks, and class attendance using Chart.js.
- **Prisma & ORM Integration**: Configured robust database migrations and schemas with PostgreSQL database engines using Prisma ORM.
- **Automated Reporting**: Integrated PDFKit library to automatically generate, style, and download student academic progress summary PDF reports.

## Tech Stack

React.js, Node.js, Express.js, PostgreSQL, Prisma, Python, Chart.js, JWT, PDFKit, Render.
