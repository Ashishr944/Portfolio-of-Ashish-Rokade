## Overview

Vera-Bot is a real-time messaging platform built to enable secure user authentication, low-latency instant communication, and scalable full-stack API integration. The application features a responsive frontend user interface and a robust backend infrastructure optimized for real-time data persistence and updates.

## Key Achievements & Implementation

- **Real-Time WebSockets**: Implemented Socket.IO based WebSocket communication to facilitate synchronized, low-latency instant messaging and status updates.
- **Secure Authentication**: Built secure signup, login, and session validation flows using JSON Web Tokens (JWT).
- **RESTful APIs**: Developed modular REST APIs to manage user data, chat histories, room creation, and user status tracking.
- **Responsive Design**: Designed a sleek interface optimized for mobile and desktop screens using Tailwind CSS.

## Tech Stack

React.js, Node.js, Express.js, MongoDB, JWT, Socket.IO, Tailwind CSS.
