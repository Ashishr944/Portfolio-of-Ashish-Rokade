## Overview

Endless Runner is an engaging 3D browser game designed to run smoothly across desktop and mobile devices. It explores real-time state calculation, mathematical physics simulation, and performant 3D model rendering directly in the client browser.

## Key Achievements & Implementation

- **Interactive 3D Graphics**: Built 3D graphics, lighting, shaders, and material rendering with Three.js and WebGL.
- **Dynamic Collision Systems**: Designed a bounding-box mathematical collision detection system for real-time interaction between player characters and hazards.
- **Smooth Inputs & Gameplay**: Created responsive player movement controllers supporting both desktop keyboard handlers and mobile-friendly tap events.
- **Procedural Obstacles**: Developed algorithms to randomly and procedurally generate path obstacles and visual scenery updates, ensuring endless gameplay variability.

## Tech Stack

JavaScript, HTML5, CSS3, Three.js, WebGL.
