## Overview

Pallawankur Nursery Android app is designed to help users shop for plants and access professional plant care guides. It features a catalogue system, item categories, cart management, and backend data services.

## Key Achievements & Implementation

- **Product Catalog Management**: Created a structured item display grid with image representation, detailed botanical descriptions, and prices.
- **Plant Care Information System**: Integrated guide pages specifying lighting, water requirements, and temperature needs for indoor and outdoor plants.
- **Secure Backend Services**: Developed database schemas and API controllers to fetch product lists, handle category sorting, and manage orders.

## Tech Stack

Java, XML, Android SDK, REST APIs, SQLite / Backend persistence.
